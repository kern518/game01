# 机甲风暴 Demo 开发台账

更新时间：2026-06-25

## Demo 闭环目标

最小可玩闭环：

1. 玩家进入地图或机库。
2. 找老师接取任务。
3. 出击并击败失控训练机。
4. 战斗结算显示奖励和下一目标。
5. 返回地图找老师交付零件。
6. 老师强化机甲。
7. 顶部目标显示 Demo 完成。

- [x] 底部 RPG 对话框替代侧边人物面板
